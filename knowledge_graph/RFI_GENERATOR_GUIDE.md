### 2. Question Extraction

Automatically finds RFI questions using patterns:

**Numbered Questions:**
```
1. What is your company's experience?
2. Describe your technical approach.
3. Provide past performance examples.
```

**Lettered Questions:**
```
A. Company background
B. Key personnel
C. Technical capabilities
```

**If no questions found:** Creates standard RFI sections

### 3. Claude AI Generation

For each question, sends a prompt to Claude:

```
You are writing an RFI response for [Opportunity].

Our Company: [Capabilities, past performance, certifications]

Question: [The RFI question]

Write a professional 2-3 paragraph response that:
- Directly answers the question
- Highlights relevant capabilities
- References specific past performance
- Uses federal contracting language
```

## ğŸ¯ Integration with Other Agents

### Combined with Opportunity Scout

```python
from opportunity_scout import OpportunityScout
from rfi_generator import RFIResponseGenerator

scout = OpportunityScout()
rfi_gen = RFIResponseGenerator()

# Get high-priority opportunities
opportunities = scout.fetch_opportunities(days_back=7)

for opp in opportunities:
    score = scout.score_opportunity(opp)
    
    # Generate RFI for high-priority with contacts
    if score['priority'] == 'HIGH' and score['contacts']['total_contacts'] > 0:
        if 'RFI' in opp.get('description', ''):
            print(f"Generating RFI for high-value opportunity: {opp['title']}")
            rfi_gen.generate_rfi_response(opp)
```

### Combined with Capability Matcher

```python
from capability_matcher import CapabilityMatcher
from rfi_generator import RFIResponseGenerator

cap_matcher = CapabilityMatcher()
rfi_gen = RFIResponseGenerator()

# Analyze capability first
cap_analysis = cap_matcher.analyze_opportunity(opportunity)

if cap_analysis['capability_score'] >= 70:
    # Only generate RFI if we can do the work
    print(f"Good capability match ({cap_analysis['capability_score']}/100)")
    print("Generating RFI response...")
    rfi_gen.generate_rfi_response(opportunity)
else:
    print(f"Poor capability match ({cap_analysis['capability_score']}/100)")
    print("Consider teaming before responding to RFI")
```
